Applying functions to collections ​

The apply family consists of functions that help apply a function f to a collection x.

In base Julia there is already the map function, but

  • It does not work on dictionaries;

  • The function is the first argument, and the collection is the second. This make it less "pipeable".

We will cover some common cases below.

One variable, one collection ​

Given a collection x and a one-variable function f, we can apply f to each element of x as follows:

julia
using TidierIteration;

x = [3:6;];
f(x) = x^2;

apply(x, f)
4-element Vector{Int64}:
  9
 16
 25
 36

This, of course, is the same as

julia
map(f, x)
4-element Vector{Int64}:
  9
 16
 25
 36

or

julia
f.(x)
4-element Vector{Int64}:
  9
 16
 25
 36

Things get more interesting when we have a dictionary as follows:

julia
d = Dict(i => i for i in [1:4;])
Dict{Int64, Int64} with 4 entries:
  4 => 4
  2 => 2
  3 => 3
  1 => 1
julia
apply(d, f)
Dict{Int64, Int64} with 4 entries:
  4 => 16
  2 => 4
  3 => 9
  1 => 1

while map(f, d) gives an error.

We can see a dictionary as a collection with named entries, and apply(d, f) means that we apply f to each value of d while keeping the keys of d intact.

In case you want to modify the keys of a dictionary, there is the special function

julia
apply_keys(d, x -> -x)
Dict{Int64, Int64} with 4 entries:
  -1 => 1
  -3 => 3
  -2 => 2
  -4 => 4

If you just want to apply f for its side-effects and return nothing, use

julia
walk(x, f)

In case you want to convert each output of f to a specific type, you can always pass a compose function:

julia
apply(x, string ∘ f)
4-element Vector{String}:
 "9"
 "16"
 "25"
 "36"

Two variables, two collections ​

We can apply a two-variable function f to two collections x and y by applying f to each pair (x_i, y_i) where x_i is the i-th element of x and y_i the i-th element of y. If x and y have different sizes, we iterate until one of them ends.

julia
using TidierIteration;

x = [1:4;]
y = [5:7;]
f(x, y) = x + y

apply2(x, y, f)
3-element Vector{Int64}:
  6
  8
 10

When x and y are dictionaries, we iterate on the set of common keys:

julia
d1 = Dict(i => i for i in [1:4;])
d2 = Dict(i => i^2 for i in [3:9;])

apply2(d1, d2, f)
Dict{Int64, Int64} with 2 entries:
  4 => 20
  3 => 12

Two variables, one collection ​

In this case, we can use the index of each element of x as the first variable to be applied on f, that is, we apply f on the pairs (i, x_i) for each index i of x. It is important to note that i is the first argument to be passed to f.

julia
using TidierIteration;

x = [3:6;]
f(i, x) = Dict(i => x)
iapply(x, f)
4-element Vector{Dict{Int64, Int64}}:
 Dict(1 => 3)
 Dict(2 => 4)
 Dict(3 => 5)
 Dict(4 => 6)

When x is a dictionary, the elements i are the keys of x:

julia
d = Dict(i => i for i in [1:4;])
g(k, v) = k + v

iapply(d, g)
Dict{Int64, Int64} with 4 entries:
  4 => 8
  2 => 4
  3 => 6
  1 => 2

One variable and one collection, dataframe output ​

When the output of f is a dataframe, we can bind all rows (or columns) quickly as follows:

julia
using TidierIteration;

x = [1:4;]
h(x) = DataFrame(:x => x)
apply_dfr(x, h)

or

julia
s = "abcd";
h2(s) = DataFrame(string(s) => rand(1))
h2("b")
apply_dfc(s, h2)

p variables and one collection ​

We can apply a p-variable function to a collection of p elements as follows:

julia
using TidierIteration
x = [
    [1, 2], [3, 4], [5, 6]
]
f(x, y, z) = x + y + z

papply(x, f)
2-element Vector{Int64}:
  9
 12

API ​

# TidierIteration.apply — Function.
julia
apply(x, f; T = identity)

Apply the function f to each element of x and return an array.

Arguments

  • x: an iterable collection.

  • f: a function.

source

julia
apply(d, f)

Apply the function f to each value of d and return a dictionary with the same keys of d.

Arguments

  • d: a dictionary.

  • f: a function.

source


# TidierIteration.apply_keys — Function.
julia
apply_keys(d, f)

Apply the function f to each key of d and return a dictionary with the same values of d.

Arguments

  • d: a dictionary.

  • f: a function.

source


# TidierIteration.walk — Function.
julia
walk(x, f)

The same as apply(x, f) but returns nothing.

source


# TidierIteration.apply_dfc — Function.
julia
apply_dfc(x, f)

Apply the function f to each element of x and bind all the columns into a dataframe.

Arguments

  • x: an iterable collection.

  • f: a function that returns a DataFrame.

source


# TidierIteration.apply_dfr — Function.

apply_dfr(x, f)

Apply the function f to each element of x and bind all the rows into a dataframe.

Arguments

  • x: an iterable collection.

# TidierIteration.iapply — Function.
julia
iapply(x, f)

Apply the function f to each pair (i, x_i) for i in the index set of x.

Arguments

  • x: an iterable collection.

  • f: a two-variable function.

source

julia
iapply(d, f)

Apply the function f to each pair (k, d[k]) for k in the values-set of d.

Arguments

  • d: a dictionary.

  • f: a two-variable function.

source

# TidierIteration.apply2 — Function.
julia
apply2(x, y, f)

Apply the 2-variable function f to each element pair (x_i, y_i) where x_i (resp. y_i) is the i-th element of x (resp. y).

Arguments

  • x: an iterable collection.

  • y: an iterable collection.

  • f: a two-variable function.

source

julia
apply2(d1, d2, f)

The same as Dict(k => f(d1[k], d2[k]) for k ∈ keys(d1) ∩ keys(d2)), that is: we apply f on (d1[k], d2[k]) for every k common key between d1 and d2.

Arguments

  • d1: a dictionary.

  • d2: a dictionary.

  • f: a two-variable function.

source


# TidierIteration.walk2 — Function.
julia
walk2(x, y, f)

The same as apply2(x, y, f) but returns nothing.

source

are elements in x.
